# Automated Currency Trend Tracker - Implementation Plan

This is a great project! Here's how to build an automated system that discovers trending currencies and tracks the news driving their movements.

## System Architecture

### Core Components

**1. Data Sources**

- **Currency data**: Free Forex API (exchangerate-api.com or currencyapi.com)
- **News & events**: Web search via Anthropic Claude API (has built-in web search)
- **Storage**: Google Sheets (free, easy to track 7-day trends)

**2. AI Analysis**

- **Claude API with web search** - analyzes news and correlates with currency movements
- Cost: ~$1-3/day for comprehensive analysis

**3. Automation**

- **Google Apps Script** (free, same as your email automation)
- Runs daily at your preferred time

## Detailed Implementation

### Phase 1: Day 1 - Discovery Phase

**What it does:**

1. Fetches current exchange rates for 50+ major currencies
2. Compares with rates from 24 hours ago
3. Identifies top 10 biggest movers (% change)
4. Searches news for each currency to understand why
5. Creates tracking sheet and sends initial report

### Phase 2: Days 2-7 - Tracking Phase

**What it does:**

1. Updates exchange rates for your tracked 10 currencies
2. Searches for latest news/events about each currency
3. Uses AI to analyze if news explains movements
4. Tracks cumulative gains/losses over 7 days
5. Sends daily summary report

### Phase 3: Day 8 - Reset

**What it does:**

1. Archives previous week's data
2. Starts new discovery cycle
3. Finds new top 10 movers
