## Full job description

Edmonton, AB (100% remote)
Work Hours: 36.25 hours per week
Standard Hours of Work: 8:15 am – 4:30pm

Project Name:
Enterprise Database Archival Service

Scope:
Reporting to the Manager, Content and Collaboration Services (CCS), the Application Analyst’s primary role will be responsible for providing technical support, troubleshooting, and system maintenance for the GoA Enterprise Database Archival Service on the Cloudera Data Platform (CDP Public Cloud). You will collaborate closely with end-users, IT teams, and stakeholders to ensure the smooth functioning of the platform. The ideal candidate will possess strong analytical skills, excellent communication abilities, and a passion for delivering exceptional customer service.

The Application Analyst will provide analysis and recommendations on process improvements and to hosting infrastructure, and include details of risks, dependencies, business impacts, interdependencies, and implications pertaining to the methodology to be applied on projects. The role serves as a bridge between users and technical teams, ensuring that applications effectively meet business objectives. A proven ability to function on an individual basis and in a team environment is essential.

Duties:

Perform routine maintenance tasks, such as software updates, patch installations, and system configurations, to ensure the stability and security of the CDP environment.
Develop and maintain technical documentations, user guides, and training materials to support users in using the CDP platform.
Work with Big Data and Business Intelligence technologies.
Understanding of CDP Public Cloud Base and Data Services from architecture level.
Support end user issues from CDP as well as infrastructure side and able to provide temporary/permanent fixes.
Work on access requests, incidents and change management process using ticketing systems like Service Now.
Should be able to administer and audit CDP User access and data management from various endpoints/levels like Console, Ranger, Virtual Warehouse, Data Visualization.
Upgrading/Patching Cloudera Data Lake, Data Hub, Data Warehouse, Virtual Warehouse, Data Visualization.
Take backups of Cloudera Data Lake, Data Hub, Data Warehouse, Virtual Warehouse, Data Visualization.
Should be able to migrate applications/data from Development to Production Environments.
Understanding, troubleshooting and fixing issues encountered in SSSD, Free IPA, Azure IDP.
Understanding, troubleshooting and fixing issues encountered in HDFS, Yarn, Ranger, Hive, Impala, Sqoop, Hue, Cloudera Data Warehouse, Cloudera Data Visualization.
Understanding, troubleshooting and fixing issues encountered in Azure Kubernetes Services, Azure Single/Flexible Postgres DB.
Should be able to manage Azure VM's, Storage Accounts, Private Subnets, Private Endpoints, Cost Explorer, Quotas, Identities/Roles.
Knowledge of managing RHEL OS, Salt orchestration is essential.
Build and optimize data warehouses, data lakes, ensuring efficient storage and retrieval system for analytical needs.
Design and create CDP DataViz dashboards and custom reporting with various data sources and inputs.
Provide depth and insight to data produced by the various teams and transform them into meaningful analytics for decision making.
Analyze data and prepare information results.
Gather and document client requirements.
Escalate issues and risks, as appropriate.
