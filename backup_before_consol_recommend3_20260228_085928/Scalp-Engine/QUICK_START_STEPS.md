# Quick Start Steps for Scalp-Engine

## Current Status
✅ OANDA credentials are configured
❌ Market state file is missing (needs to be generated by Trade-Alerts)

## Step-by-Step Instructions

### Step 1: Generate Market State File (Required First)

The Scalp-Engine needs `market_state.json` from Trade-Alerts. Generate it:

```powershell
# Go to Trade-Alerts directory
cd ..\Trade-Alerts

# Run Trade-Alerts to generate market_state.json
python main.py
```

**What to expect:**
- Trade-Alerts will run its analysis
- It will generate `market_state.json` in the Trade-Alerts folder
- Wait for it to complete (may take a few minutes)

**To check if it's done:**
```powershell
# In Trade-Alerts directory
Test-Path market_state.json
# Should return: True
```

### Step 2: Return to Scalp-Engine

```powershell
# Go back to Scalp-Engine
cd ..\Scalp-Engine
```

### Step 3: Verify State File Exists

```powershell
# Check if state file is now available
Test-Path ..\Trade-Alerts\market_state.json
# Should return: True
```

### Step 4: Run Scalp-Engine

```powershell
# Now run the scalper
python scalp_engine.py
```

## What You'll See

### If State File Doesn't Exist:

## Alternative: Create a Test State File

If you want to test Scalp-Engine without running full Trade-Alerts:

```powershell
# Create a test market_state.json
cd ..\Trade-Alerts

# Create test file
@"
{
    "timestamp": "2026-01-09T10:00:00Z",
    "global_bias": "BULLISH",
    "regime": "TRENDING",
    "approved_pairs": ["EUR/USD", "USD/JPY"],
    "long_count": 2,
    "short_count": 0,
    "total_opportunities": 2
}
"@ | Out-File -FilePath market_state.json -Encoding utf8

# Go back to Scalp-Engine
cd ..\Scalp-Engine
```

## Troubleshooting

### If Scalp-Engine hangs on "Waiting for Trade-Alerts..."
- The state file doesn't exist or is stale (> 4 hours old)
- Solution: Run Trade-Alerts to generate a fresh state file

### If you get OANDA API errors
- Check your token is correct (no extra spaces)
- Verify account ID matches your OANDA account
- Ensure environment is "practice" for testing

### To Stop Scalp-Engine
Press `Ctrl + C`
