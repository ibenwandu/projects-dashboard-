# Position Sizing One-Page Quick Start

## THE FORMULA (Memorize This)

```
Position Units = (Account Balance × Risk %) / (Stop Loss Pips × Pip Value)

Example: ($100 × 0.005) / (10 × 0.0001) = 500 units
```

## RULES (NON-NEGOTIABLE)

1. ✅ **EVERY trade must have a stop loss** (100% coverage)
2. ✅ **Same position size every trade** (no emotion-based sizing)
3. ✅ **Stop after 2% daily loss** (prevents revenge trading)
4. ✅ **Stop after 5 consecutive losses** (circuit breaker)
5. ✅ **Use fixed fractional risk** (formula above)

```

Pair: EUR/USD

TP: 1.0870 (20 pips, 1:2 ratio)

Position = ($100 × 0.005) / (10 × 0.0001) = 500 units

If stopped: Lose $0.50 ✅
If target hit: Win $1.00 ✅
R:R Ratio: 1:2 ✅
```

## DAILY TRADING FLOW

```
1. Morning: Check account balance & reset daily P&L counter
2. Pre-Trade: Verify consensus ≥2, price age <100 pips, no daily loss yet
3. Calculate: Position = (Account × 0.5%) / (SL × pip_value)
4. Place: Entry + Stop Loss + Take Profit
5. Monitor: Track daily P&L
6. Stop: After 2% daily loss OR 5 consecutive losses
7. Log: Record entry, exit, P&L, reason
8. End of Day: Update account balance, review P&L
```

---

## WHEN TO STOP TRADING (That Day)

🛑 **STOP immediately if**:
- Daily loss reaches 2% of account ($2.00 on $100)
- You have 5 consecutive losses
- Stop loss is not executing properly
- System error or connectivity issue
- You're feeling emotional or revenge-trading

## COMMON MISTAKES (NEVER DO)

| Mistake | Why It Kills Accounts | Solution |
|---------|----------------------|----------|
| No stop loss | Can lose 100x account in 1 trade | Mandatory on every trade |
| Increase size after loss | Doubles losses when emotional | Fixed position sizing always |
| Ignore daily limits | One bad day becomes catastrophic | Stop after 2% daily loss |
| Over-leverage | Tiny move triggers margin call | Use 1:1 leverage + position sizing |
| Wrong R:R ratio | Win less often than lose | Target 1:2 minimum |
